Electronic notebook for "Empirically derived thermal thresholds of four coral species along the Red Sea using a portable and standardized experimental approach", published in Coral Reefs (2022), accessible open access here: https://link.springer.com/article/10.1007/s00338-022-02233-y

Contains two folders: one with the main figures and one with the supplementary materials

Each Folder contains a PDF of the figure, the data and R code used to produce the figure, and the R code detailing how the statistical analyses were conducted (where applicable).
